/84/system/alert

GET

This call queries for a list of All Alert ids across all symmetrix arrays

Parameters

name description type default
type Optional value that filters returned list to display Alert Ids that has the following severity: ARRAY, PERFORMANCE, SERVER query
description Optional value that filters returned list to display Alerts that have the specified description e.g. equal to "description=description1" or contains "description=< like>description" query
severity Optional value that filters returned list to display Alert Ids that has the following severity: NORMAL, INFORMATION, MINOR, WARNING, CRITICAL, FATAL query
state Optional value that filters returned list to display Alert Ids that has the following state: NEW, ACKNOWLEDGED, CLEARED query
created_date Optional value that filters returned list to display Alert Ids that are greater than(">1"), Less than("<1") or equal to the specified created_date "MMM-dd-yyyy HH:mm:ss.SSS" query
created_date_milliseconds Optional value that filters returned list to display Alert Ids that contain greater than(">1"), Less than("<1") or equal to the specified date in milliseconds query
acknowledged Optional value that filters returned list to display Alert Ids that are acknowledged or not (true/false) query

Response Body

element: listAlertResult
media types: application/xml
application/x-java-serialized-object
application/json

A list of Alert Ids
