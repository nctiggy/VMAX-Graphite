/82/wlp/symmetrix/{symmetrixId}/headroom

GET

Get headroom capacity values for all SLO/WorkloadType combinations on all SRPs on a given Symmetrix array, Headroom expiry with confidence > 0.5 will also be returned.

Parameters

name description type default
symmetrixId The Symmetrix Array ID path
srp Optional value that filters the returned list to only show specified SRP Headroom details query
slo Optional value that filters the returned list to only show specified slo Headroom details query
workloadtype Optional value that filters the returned list to only show specified workloadtype Headroom details [NONE,OLTP,OLTP_REP,DSS,DSS_REP] query
emulation Optional value that filters the returned list to only show headroom for a specific emulation for an SRP [FBA, CKD] query
headroom_expiry Optional value that forces Headroom expiry with confidence <0.5 to be displayed (true/false) query

Response Body

element: wlpHeadroomResult
media types: application/xml
application/x-java-serialized-object
application/json

List of headroom capacity values and last updated timestamps.

/82/wlp/symmetrix/{symmetrixId}/utilization

GET

Returns the most utilized component and utilization value for each 4 hour time window over the most recently processed two weeks.

Parameters

name description type default
symmetrixId - The Symmetrix Array ID. path
bucket - Optional filter overrides normal behavior. Returns up to 2 utilization values for all components during the specified 4 hour bucket index. query
headroom - Optional filter (default false) to ignore components that do not impact headroom calculations. query false

Response Body

element: maxArrayComponentUtilizationList
media types: application/xml
application/x-java-serialized-object
application/json

Component utilizations.
