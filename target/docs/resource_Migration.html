/83/migration/symmetrix/{symmetrixId}/storagegroup

GET

This call gets Storage Group names for a Symmetrix.

Parameters

name description type default
symmetrixId the Symmetrix that the request is being made to. This is the 12 digit Symmetrix ID. path
includeMigrations optional parameter also add a list of migrating SG names. True means SGs with migrations will be included query

Response Body

element: dmStorageGroupList
media types: application/xml
application/x-java-serialized-object
application/json

the object containing the list of Storage Group names
