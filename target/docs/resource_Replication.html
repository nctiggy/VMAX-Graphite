DELETE

This call terminates a snapshot generation. If the snapshot generation is restored, the restored session will be terminated. If there is no restore session associated with the snapshot generation, the generation itself will be terminated. If it is not known if the generation is restored before issuing the DELETE command, performing a GET for the generations after a successful DELETE will indicate if the generation still exists (a restore was terminated) or if the generation is gone (the original generation was terminated).

Parameters

name description type default
symmetrixId the Symmetrix that the Storage Group is on. This is the 12 digit Symmetrix ID. path
storageGroupId the Storage Group name for the group being acted upon path
snapshotId the snapshot name being acted upon path
generationNumber the snapshot generation being acted upon path
force (no documentation provided) query
symforce (no documentation provided) query
star (no documentation provided) query
